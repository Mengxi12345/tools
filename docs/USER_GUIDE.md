# 用户使用手册

## 快速开始

### 1. 登录系统

访问系统首页，使用用户名和密码登录。

### 2. 添加平台

1. 进入「平台管理」页面
2. 点击「添加平台」
3. 填写平台信息（名称、类型、配置）
4. 点击「测试连接」验证配置
5. 保存平台

### 3. 添加用户

1. 进入「用户管理」页面
2. 点击「添加用户」
3. 选择平台和输入用户信息
4. 设置优先级和标签（可选）
5. 保存用户

### 4. 拉取内容

1. 在用户列表中，点击「刷新」按钮
2. 选择时间范围（默认从上次拉取时间至今）
3. 等待拉取完成
4. 查看拉取结果

### 5. 查看内容

1. 进入「内容管理」页面
2. 使用搜索和筛选功能查找内容
3. 点击内容标题查看详情
4. 标记已读或收藏感兴趣的内容

## 功能说明

### 平台管理

- **添加平台**: 支持添加多个内容平台（GitHub、Medium、Reddit 等）
- **平台配置**: 每个平台可以配置不同的认证信息
- **连接测试**: 验证平台配置是否正确

### 用户管理

- **添加用户**: 添加要追踪的平台用户
- **用户分组**: 将用户分组管理
- **优先级设置**: 设置用户优先级，影响内容排序
- **标签管理**: 为用户添加标签，方便分类和筛选

### 内容管理

- **内容列表**: 查看所有拉取的内容
- **搜索功能**: 支持关键词搜索、正则表达式搜索、高级搜索
- **筛选功能**: 按平台、作者、类型、标签筛选
- **内容操作**: 标记已读、收藏、删除

### 数据分析

- **统计概览**: 查看内容统计、平台分布、用户统计
- **图表展示**: 可视化展示数据趋势和分布
- **时间分析**: 分析内容发布时间分布和增长趋势

### 导出功能

- **格式支持**: 支持 JSON、Markdown、CSV、HTML 格式
- **筛选导出**: 可以按用户筛选导出内容
- **异步导出**: 大数据量支持异步导出

### 通知管理

- **通知规则**: 配置内容通知规则
- **通知类型**: 支持邮件、Webhook、桌面通知
- **免打扰时段**: 设置免打扰时段，避免打扰

## 常见问题

### Q: 如何添加新的平台？

A: 进入「平台管理」页面，点击「添加平台」，填写平台信息并配置认证信息。

### Q: 内容拉取失败怎么办？

A: 检查平台配置是否正确，网络连接是否正常，以及平台 API 是否可用。

### Q: 如何批量导出内容？

A: 进入「导出」页面，选择导出格式和筛选条件，点击导出。

### Q: 如何设置定时拉取？

A: 进入「设置」页面，开启全局定时任务或为特定用户开启定时任务。

### Q: 搜索功能如何使用？

A: 在内容管理页面，可以使用搜索框进行关键词搜索，或点击「高级搜索」使用正则表达式和高级筛选。

## 快捷键

- `Ctrl + K`: 快速搜索
- `Ctrl + F`: 页面内搜索
- `Esc`: 关闭弹窗
